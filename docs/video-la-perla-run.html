La Perla Run x Calistenia App

Guia de grabacion para el equipo

Guillermo corre La Perla Run mientras su app trackea via GPS. A mitad de carrera, para y le habla a una IA por voz desde su telefono — la IA escribe el codigo de un feature personalizado para la carrera. Hace deploy en vivo, termina la carrera, y muestra el resultado: una share card con "LA PERLA RUN" y sus stats reales.

1 Concepto del Video

"Le hable a una IA mientras corria y ella codeo un feature para mi app"

Inspirado en un video viral donde alguien codifico una app mientras corria un maraton usando dictado por voz. Nuestro twist es MEJOR: Guillermo corre La Perla Run, para en el km 2, y le habla a una IA (Claude Code) por voz desde su telefono conectado via Tailscale a su PC de escritorio. Le dice lo que necesita, la IA escribe el codigo, y el hace deploy — todo durante la carrera. Al terminar muestra el resultado: una share card personalizada con "LA PERLA RUN" y sus stats reales.

Duracion: 60-75s Formato: Vertical 9:16 Para: Reels / TikTok Tono: Absurdo + tecnico

🤖 Por que esto es mejor que el video original

  • 💬 El original: un pana dictaba texto a codigo con Wispr Flow (glorified dictado)
  • 🤖 Nuestro video: la IA PIENSA y genera el codigo — Guillermo solo le dice que quiere
  • 📱 Setup: telefono con Termius + Tailscale + Claude Code CLI en su PC de escritorio
  • 🎤 Voz: dictado de iOS (el microfono del teclado) transcribe lo que dice directo en la terminal
  • 🚀 Resultado: un feature REAL desplegado en produccion durante una carrera

⚙️ Setup Tecnico (como funciona)

Diagrama de conexion

🎤 Voz de Guillermo
    ↓
📱 iPhone (dictado iOS → texto en teclado)
    ↓
🔒 Termius app (SSH via Tailscale VPN)
    ↓
🖥 PC de escritorio (tmux + Claude Code CLI)
    ↓
🤖 Claude Code genera el codigo
    ↓
🚀 git push → deploy en produccion

Preparacion antes de la carrera

  • 1️⃣ En tu PC: Abrir terminal, iniciar tmux new -s race
  • 2️⃣ En tmux: Navegar al repo y tener Claude Code listo (cd calistenia-app && claude)
  • 3️⃣ Tailscale: Confirmar que esta activo y tu PC es alcanzable desde el telefono
  • 4️⃣ En el iPhone: Abrir Termius, conectar via SSH a tu PC (usar IP de Tailscale)
  • 5️⃣ Probar: Hacer tmux attach -t race y verificar que ves Claude Code
  • 6️⃣ Probar dictado: Presionar el microfono del teclado iOS y hablar — verificar que el texto aparece en la terminal
  • ⚠️ Screen recording: Activar en AMBOS — telefono (iOS) y PC (OBS o QuickTime)

El truco de produccion

🎬 Plan A + Plan B

Plan A (ideal): Durante la carrera, hablarle a Claude Code EN VIVO. Si responde bien y el internet funciona — perfecto, usas esa toma.

Plan B (backup): Pre-grabar la sesion de Claude Code en tu PC la noche anterior. Durante la carrera, solo grabas la parte de VOZ + la terminal abierta. En edicion combinas: tu voz real + screen recording de Claude trabajando (pre-grabado) + deploy real.

El deploy SI es real: En ambos planes, el git push durante la carrera es genuino. El feature se despliega en vivo.
